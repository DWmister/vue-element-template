# vue3 
>`master`为功能集成模块
>
>`vue3-ts-template`分支为基础模块
>
>`vue3-ts-pure`分支为初始化项目，暂无任何功能

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
